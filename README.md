# NotePlace

ノートを置く場所を提供するサービス。公開範囲を指定することで簡単かつ安全にノートを公開することができる。

## 機能

- ノート(pdf or markdown)をアップロード
- ノートの公開範囲を指定
- ユーザーのメール認証
- 所属する団体のノートをタイムラインで表示
